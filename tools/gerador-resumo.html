Como usar o Gerador de Resumos?

  • 1. Cole seu texto: Coloque o conteúdo original na área de "Texto de Origem".
  • 2. Escolha o tamanho: Use a barra azul para decidir se quer um resumo bem curto ou com mais detalhes.
  • 3. Processe: Clique no botão "Criar Resumo Agora" e nossa ferramenta analisará o texto para você.
  • 4. Finalize: Veja o resultado ao lado, copie para seu trabalho ou confira os assuntos principais detectados automaticamente.
